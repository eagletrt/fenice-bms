Function Documentation¶

EepromOperations m95256_WriteBuffer(m95256_t eeprom, uint8_t *pBuffer, uint16_t WriteAddr, uint16_t NumByteToWrite)

Writes block of data to the EEPROM. In this function, the number of WRITE cycles are reduced, using Page WRITE sequence.

Parameters
  • eeprom: EEPROM instance handle
  • pBuffer: pointer to the buffer containing the data to be written to the EEPROM.
  • WriteAddr: EEPROM’s internal address to write to.
  • NumByteToWrite: number of bytes to write to the EEPROM.
Return Value
  • EepromOperations: value: EEPROM_STATUS_COMPLETE or EEPROM_STATUS_ERROR
