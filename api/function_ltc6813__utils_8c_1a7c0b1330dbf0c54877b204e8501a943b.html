size_t ltc6813_read_voltages(SPI_HandleTypeDef *hspi, voltage_t *volts)¶

Polls all the registers of the LTC6813 and updates the cell array.

It executes multiple rdcv requests to the LTCs and saves the values in the voltage variable of the CELL_Ts.
